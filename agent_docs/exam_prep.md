# WAEC/JAMB Prep Module

## Description

Exam-focused preparation system with past questions and mock exams.

## Requirements

* Past questions database
* Exam simulation (timed)
* Topic-based practice
* Result breakdown

## Endpoints

* `GET /exams`
* `GET /exams/{id}/questions`
* `POST /exams/submit`

## Tools

* PostgreSQL
* R2 for large datasets
