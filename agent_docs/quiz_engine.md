# Quiz & Assessment Engine

## Description

Generates and delivers quizzes with feedback and scoring.

## Requirements

* Multiple choice questions
* Instant feedback
* Score tracking
* Adaptive difficulty
* Offline quiz support

## Endpoints

* `POST /quiz/generate`
* `GET /quiz/{id}`
* `POST /quiz/submit`

## Tools

* Django + DRF
* PostgreSQL
* Optional AI for generation
